08. In Bootstrap 4, what is flexbox?

In Bootstrap 4, Flexbox (Flexible Box Layout) is a powerful layout model used for designing flexible and responsive page layouts. Bootstrap 4 heavily relies on Flexbox to create a grid system that allows for easy alignment and distribution of space among elements within a container.

Key features of Flexbox in Bootstrap 4 include:

  1. Grid System:
    • Bootstrap 4's grid system is built on top of Flexbox. The display: flex property is applied to the container, and the child elements become flex items. This allows for the creation of responsive and flexible layouts.
  2. Flex Containers and Items:
    • Elements with the class d-flex become flex containers, and their children become flex items. This enables developers to control the layout, alignment, and spacing of items within the container using Flexbox properties.
  3. Flex Utilities:
    • Bootstrap 4 provides a set of utility classes for working with Flexbox. These classes include utilities for adjusting container properties (d-flex, d-inline-flex), item alignment and ordering (align-items, align-self, order), and flex direction (flex-row, flex-column, flex-row-reverse, flex-column-reverse).
  4. Justify Content and Align Items:
    • Flexbox in Bootstrap 4 allows developers to easily justify content and align items within a flex container. Utilities like justify-content and align-items can be used to control the horizontal and vertical alignment of items.
  5. Responsive Flex Utilities:
    • Flexbox utilities in Bootstrap 4 are responsive, meaning you can apply different Flexbox behavior based on screen size breakpoints. For example, you can use d-md-flex to apply the flex display only on medium-sized screens and larger.
