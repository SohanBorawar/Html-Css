10. What is a bootstrap card and how would you create one?

A Bootstrap card is a flexible and lightweight container used to display various types of content. Cards in Bootstrap provide a unified structure for presenting information, such as text, images, links, and buttons, in a visually appealing way. Cards are responsive and can be easily customized to fit different design needs.

In this example:

  • The outer container "div class="card"" represents the card itself.
  • "img" is used for the card's image. You can replace the src attribute with the URL of your image.
  • Inside the "div class="card-body"", you can include the main content of the card, such as a title "h5", text "p", and a button "a".

You can further customize the appearance of the card by using additional Bootstrap classes or your own styles. Bootstrap provides a variety of card-related classes for different purposes, such as card-header, card-footer, card-subtitle, etc. Explore the Bootstrap documentation for more advanced customization options: https://getbootstrap.com/docs/4.5/components/card/
